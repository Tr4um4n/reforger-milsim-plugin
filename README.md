# Arma Reforger MILSIM Management Plugin

Plugin nativo para WordPress que gestiona misiones, eventos, ORBAT y condecoraciones para clanes de Arma Reforger.

## Características
- Gestión de Misiones y Eventos.
- Motor de ORBAT interactivo con JSON.
- Sistema de Roles y Capacidades MILSIM.
- Pasador de Medallas (Ribbon Rack) automatizado.
- Sincronización con Workshop API.

## Instalación
1. Sube la carpeta al directorio `wp-content/plugins/`.
2. Activa el plugin desde el panel de administración.
